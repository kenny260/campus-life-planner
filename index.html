About Campus Life Planner

Purpose

Campus Life Planner is a comprehensive task management application designed specifically for students to organize their academic life. Track assignments, manage study sessions, plan group meetings, and stay on top of your campus activities.

Features

  • Create, edit, and delete tasks with detailed information
  • Advanced regex search with pattern highlighting
  • Visual dashboard with statistics and charts
  • Organize tasks with customizable tags
  • Mark tasks as complete/incomplete
  • Sort and filter by multiple criteria
  • Export and import your data as JSON
  • Keyboard shortcuts for power users
  • Full accessibility support
  • Responsive design for all devices

Keyboard Shortcuts

Ctrl + F Focus search box
Ctrl + N Create new task
Esc Cancel editing
